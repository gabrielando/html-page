Breve historia da Web:

(Voltar)

A Web (World Wide Web) foi criada em 1989 por Tim Berners-Lee, com o objetivo de facilitar o compartilhamento de informações entre cientistas. Inicialmente, era apenas um sistema de hipertextos acessado por navegadores simples. Com o tempo, evoluiu para incluir imagens, vídeos, interatividade e aplicações complexas, tornando-se parte essencial do nosso dia a dia.

Como Surgiu:

A ideia da Web surgiu da necessidade de compartilhar documentos científicos entre universidades. Tim Berners-Lee, trabalhando no CERN, propôs um sistema que usava links (hipertexto) para conectar informações, acessadas por meio de navegadores. Em 1991, a Web foi lançada publicamente e começou a se espalhar pelo mundo.

Primeiros Computadores:

Os primeiros computadores surgiram na década de 1940. Um dos mais conhecidos foi o ENIAC, criado em 1946 nos EUA. Eram máquinas enormes, com milhares de válvulas, usadas principalmente para cálculos militares e científicos. Esses computadores ocupavam salas inteiras e consumiam muita energia.

Grande Nomes:

  1. Tim Berners-Lee - Criador da Web.
  2. Alan Turing - Pioneiro da computação, idealizador do conceito de máquina universal
  3. Ada Lovelace - Considerada a primeira programadora da história.
  4. Bill Gates - Fundador da Microsoft, ajudou a popularizar o computador pessoal.
  5. Steve Jobs - Co-fundador da Apple, contribuiu para tornar a tecnologia mais acessível e amigável ao usuário.

Hardware e Software:

Hardware é a parte física do computador, ou seja, todos os componentes que você pode ver e tocar. Ele é responsável por executar as instruções dos programas e realizar tarefas. Os principais tipos de hardware incluem:

  • Placa-mãe: é a base que conecta todos os outros componentes.
  • Processador (CPU): o “cérebro” do computador, que realiza os cálculos e comandos.
  • Memória RAM: memória temporária usada para rodar programas em tempo real./li>
  • HD/SSD: responsáveis por armazenar arquivos e programas.
  • Periféricos: teclado, mouse, monitor, impressora, etc.
Sem o hardware, o computador não funcionaria, pois é ele quem executa as ações que o software comanda.

Software é a parte lógica do computador, formada pelos programas, aplicativos e sistemas operacionais que dão instruções ao hardware sobre o que fazer. Existem dois tipos principais:

  • Software de sistema: como o sistema operacional (ex: Windows, Linux, macOS), que gerencia todo o funcionamento do computador.
  • Software de aplicação: como navegadores (Google Chrome), editores de texto (Word), jogos, redes sociais, etc.
O software depende do hardware para funcionar, e o hardware depende do software para saber o que fazer. Juntos, eles tornam possível o uso de computadores e dispositivos eletrônicos

Linguaguem de Programação

(Voltar)

O que é uma Linguagem de Programação? Uma linguagem de programação é uma ferramenta que usamos para dar instruções a um computador. É como um idioma que humanos usam para "conversar" com máquinas e dizer o que elas devem fazer, ela permite escrever algoritmos, que são sequências de passos lógicos para resolver problemas ou executar tarefas.

Principais características de uma linguagem de programação:

  • Sintaxe: Regras que definem como o código deve ser escrito. Ex: ponto e vírgula no final de uma linha, uso de chaves, palavras-chave como if, else, function.
  • Semântica: Significado das instruções escritas. Por exemplo, o que acontece quando você escreve x = x + 1.
  • Lógica de controle: Permite tomar decisões e repetir ações: if, else, for, while.
  • Variáveis e Tipos de Dados: Usadas para armazenar informações. Exemplos de tipos: números, textos (strings), listas, booleanos (verdadeiro/falso).
  • Funções ou Métodos: Blocos de código que podem ser reutilizados para executar tarefas específicas.

HTML (HyperText Markup Language) - HTML é uma linguagem de marcação

É a linguagem base da web, ele estrutura o conteúdo de uma página, organizando textos, imagens, links, vídeos, botões, etc. HTML não é uma linguagem de programação, embora o HTML seja essencial para criar páginas web, ele não é considerado uma linguagem de programação. O próprio nome diz: HTML = HyperText Markup Language, ou seja, linguagem de marcação de hipertext. Isso significa que o HTML marca e organiza o conteúdo da página (textos, imagens, links), mas não executa comandos nem faz cálculos.

Tags HTML – O que são?

As tags são elementos do HTML. Elas indicam ao navegador o tipo de conteúdo ou a função daquele trecho.

Principais Tags HTML (com explicação e exemplo)
      <html> = Define o início de um documento HTML.
      <head> = Contém informações da página que não aparecem diretamente no conteúdo visual, como título, meta dados e links para arquivos externos (CSS, fontes, etc).
      <title> = Define o título da aba do navegador.
      <body> = É o corpo da página, onde vai o conteúdo visível (texto, imagens, vídeos).
      <h1> = a <h6> Usado para títulos e subtítulos, do mais importante <h1> para o menos importante <h6>.
      <p> = Define um parágrafo de texto.
      <a> = Cria um link.
      <img> = Exibe uma imagem, atributo src define a fonte da imagem e alt é o texto alternativo.
      <ul> <ol> <li>= Listas: <ul> = lista não ordenada (com marcadores). <ol> =  lista ordenada (com números). <li> = item da lista.
      <br> = Cria uma quebra de linha (como apertar Enter).
      <strong> <b> = Texto em negrito: <strong> dá ênfase semântica (importância). <b> só deixa o testo em negrito visualmente
      <hr> = Insere uma linha divisória horizontal na página.
      <em> <i> = Texto em itálico: <em> dá ênfase semântica (destacar significado). <i> apenas deixa o testo em itálico visualmente.
